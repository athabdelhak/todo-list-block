# todo-list-block

TodoMass: Dynamic Day Architect TodoMass is a sleek, lightweight web application designed for high-efficiency daily planning. Specifically tailored for ENSTI students, it utilizes a Time-Blocking methodology to help users visualize their day, 1 pixel at a time.

Key Features Visual Timeline: A dynamic schedule where 1.5 pixels equals 1 minute, providing a literal "view" of your time.

Live Day Progress: A real-time progress bar that shows exactly how much of your day has passed.

Intelligent Clock: A digital clock that automatically updates the day of the week, month, and date.

Task Management: Add, edit, and delete activities with color-coded tags for better organization.

Conflict Detection: Built-in logic to prevent overlapping tasks, ensuring a realistic schedule.

Dual Themes: Full support for Dark Mode and Light Mode for comfortable viewing day or night.

Mobile Optimized: Fully responsive design with touch-scroll support for planning on the go.

Remix Icon: Elegant vector iconography.

Google Fonts: "Outfit" typography for a clean, professional aesthetic.

Mobile Experience The application includes a specialized media query fix to ensure that the timeline remains fluid and scrollable on smartphones, resolving common overflow issues found in standard web apps.
